Student Management System

A comprehensive platform that enables teachers to manage student marks efficiently and allows students to access their academic performance with ease.

Get Started

Key Features

Teacher Portal

Teachers can easily add, update, and manage student details and academic marks through an intuitive interface.

Student Access

Students can securely view their marks and track their academic progress throughout the semester.

RESTful API

Powerful API endpoints for seamless integration with other educational systems and applications.

Technical Overview

Backend

Built with Flask framework, SQLite database, and Flask-SQLAlchemy ORM for robust data management.

Frontend

Responsive user interface developed using HTML, CSS, and JavaScript for optimal user experience.

Installation

Easy setup process with comprehensive documentation for quick deployment and configuration.
